## Key Features of the Race Targets Section:

1. **Individual Cards for Each Race-Gender Combination**
   - Each card shows the target demographic (e.g., "African Female")
   - Displays current count vs. target (e.g., "150 / 435")
   - Shows progress percentage (e.g., "34.5%")
   - Progress bar is color-coded:
     - Red: < 75% of target
     - Yellow: 75-99% of target
     - Green: ≥ 100% of target

2. **Enhanced Race Distribution Chart**
   - Changed from pie chart to stacked bar chart
   - Each bar represents a race category
   - Bars are stacked to show gender breakdown
   - Female segments shown in pink
   - Male segments shown in blue

3. **Responsive Layout**
   - Cards arranged in rows of 3 on large screens
   - Adapts to fewer columns on smaller screens
   - Maintains consistent styling with other dashboard elements

This mockup shows how the race-gender targets integrate seamlessly with the existing dashboard design while providing valuable additional information for diversity tracking.
